# HAMOOPIG

A **Sega Genesis\Mega Drive** engine for *fighting games*!

## Sobre o Projeto

**HAMOOPIG** é a implementação da engine **HAMOOPI** no console **SEGA Genesis/Mega Drive**. Uma engine de jogo de luta open source que permite criar jogos tão bons quanto aqueles clássicos jogos de arcade dos anos 80/90!
